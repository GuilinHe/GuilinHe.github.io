SQL语句

SQL 语句

1.查询数据 select

  1. 从指定表中查找所有列的数据: SELECT * FROM 表名称

  2. 从指定表中查找指定列的数据:SELECT 列名称 FROM 表名称

2.插入数据 insert

向指定的数据表中插入一条数据: INSERT INTO 表名称(列1, 列2, ...) VALUES(值1, 值2, ...)

注意: 多个列和值之间使用逗号隔开, 列和值的顺序要保持一致

3.更新数据 update

更新数据表中的某一列或多列数据: UPDATE 表名称 SET 列1=值1, 列2=值2 WHERE 列名称=值;

注意: SET 后多列数据更新使用逗号隔开, WHERE 后面跟的条件不能省略

4.删除数据 delete

删除数据表中的某一行或多行数据： DELETE FROM 表名称 WHERE 列名称=值;

5.where 子句的作用

如果需要有条件的从表中选取数据, 可以使用 WHERE 子句添加到 SQL 语句的末尾

例如:

SELECT * FROM 表名称 WHERE 列名称 运算符 值

常见的运算符：

(1) > 和 >=

(2) < 和 <=

(3) != 或 <>

(4) =

(5) BETWEEN

(6) LIKE

AND 和 OR 运算符

可以在 WHERE 子句中把多个条件结合起来, AND 表示并且, OR 表示或者的意思

6.order by 子句的作用

  1. 使用 ORDER BY 子句对结果进行排序

    SELECT * FROM users ORDER BY status DESC;

默认的排序方式为升序 ASC, 如果使用降序可以添加 DESC 关键字

  1. 多重排序规则

    SELECT 列名称 FROM 表名称 ORDER BY 列名称 [排序规则], 列名称 [排序规则]

  1. 注意

    同时使用 order by 和 where子句时, 应该让 order by 位于 where 之后, 否则将会产生错误
