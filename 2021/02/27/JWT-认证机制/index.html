JWT 认证机制

JWT 认证机制

  • 基本概念

    JWT（英文全称：JSON Web Token）是目前最流行的跨域认证解决方案。

  • 工作原理

    工作原理总结：用户的信息通过 Token 字符串的形式，保存在客户端浏览器中。服务器通过还原 Token 字符串的形式来认证用户的身份。

  • 安装 JWT 相关的包

    同时安装两个JWT相关的包

    • jsonwebtoken 用于生成 JWT 字符串

    • express-jwt 用于将 JWT 字符串解析还原成 JSON 对象
